=== КАК ПОМЕНЯТЬ ВСЁ НА УЖЕ СОЗДАННОМ КАНАЛЕ ===
Почти всё — в одном месте: studio.youtube.com -> слева "Customization / Настройка канала".
Там 3 вкладки: LAYOUT (макет) / BRANDING (фирстиль) / BASIC INFO (осн. инфо).
ВАЖНО: после правок жми "PUBLISH / Опубликовать" (вверху справа) — иначе не сохранится.

1) АВАТАР
   Customization -> BRANDING -> Picture -> Change -> загрузи avatar.png (800x800) -> подгони круг -> Done -> PUBLISH.

2) ШАПКА (баннер)
   BRANDING -> Banner image -> Change -> загрузи banner.png (2560x1440) -> проверь, что текст в "safe area" -> Done -> PUBLISH.

3) ВОДЯНОЙ ЗНАК НА ВИДЕО
   BRANDING -> Video watermark -> Upload -> watermark.png -> Display time: "Entire video" -> PUBLISH.

4) НАЗВАНИЕ КАНАЛА
   BASIC INFO -> Name -> впиши  PolarStar Lab  -> PUBLISH.   (лимит: до 3 смен имени за 14 дней)

5) ХЭНДЛ (@)
   BASIC INFO -> Handle -> впиши  polarstarlab  -> youtube.com/@polarstarlab -> PUBLISH.

6) ОПИСАНИЕ
   BASIC INFO -> Description -> вставь описание (см. выше) -> PUBLISH.

7) ССЫЛКИ + КОНТАКТ
   BASIC INFO -> Links -> Add link -> Website -> polarstarlab.com.
   BASIC INFO -> Contact info / Email -> hello@polarstarlab.com.

8) КЛЮЧЕВЫЕ СЛОВА + СТРАНА
   Шестерёнка "Settings" (внизу слева) -> Channel -> Basic info -> Keywords (вставь список) + Country = United States.

ПОМНИ:
- Старые видео и подписчики остаются — ты только переодеваешь обложку канала.
- Аватар/баннер обновляются не мгновенно — до нескольких часов.
- Имя меняется ограниченно (3 раза/14 дней) — не дёргай туда-сюда.
- Если канал "личный" (по твоему Google-аккаунту) — переименование в Customization всё равно работает.
