# PolarStar Lab — launch & brand

Brand assets, content, and the public landing page for **PolarStar Lab** — a trading
strategy and research lab for **NinjaTrader 8** built on the liquidity-and-fractals method.

**Live site:** https://sergeisysoev.github.io/polarstarlab/ (custom domain: polarstarlab.com)

## Contents
- `index.html` — the landing page (self-contained, no build step, mobile-responsive).
- `brand/` — avatar, channel banner, video watermark, logo-style explorations, channel + rebrand texts.
- `video/` — the YouTube Short (final mp4) and all scripts: short script, full course storyboard, clean narration, YouTube metadata.
